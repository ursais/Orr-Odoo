Instruction and Quality Checklist on Job Order

This app allow you to create instructions and quality checklists for job order / work order.

Main Features:

- Allow you to configure instructions related to your job orders.
- Allow you to configure quality checklist related to your job orders.
- Allow project manager / job manager to set list of instructions to any job order.
- Worker/Technicians can see instruction and work accordingly.
- Allow you to set image/photo on instruction master data.
- Workflow on Instruction of job order Start-Pause-Finished.
- Print instruction pdf with photos.
- For more details: See Video in live preview

Available Menus

  • Jobs/Configuration/Quality Checklists
