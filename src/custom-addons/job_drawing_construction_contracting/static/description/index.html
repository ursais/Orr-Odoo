Main Features:

  • This modules allow project manager and project user to add Drawings document on Job Order form.
  • You can add your google drawings links on Job order form. You can add multiple drawing links on job order form.

  • Allow your technician / worker / site manager / Contractor to edit/change google drawings on website / My account / Portal.
  • For every google drawing on every job order has separate and unique page.
