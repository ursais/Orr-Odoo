Employee Expense Integration Job Orders for Contruction and Contract Project

This app allow your employees who is working into job order or work order in system to submit expense with pictures of expense.

Main Features:

- Allow your employee to submit expense for job order they are working.
- On Expense form employee can select project, job order and photos for expense.
- Related smart buttons on Job order and Project to have quick view.
