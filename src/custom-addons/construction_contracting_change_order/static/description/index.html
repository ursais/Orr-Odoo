Change Order in Job Contracting and Construction

This module allow you to manage Change Order in Job Contracting and Construction Business.

Main Features:

- Allow Project/Site/Sales Users to create Change Order.
- Approval Workflow : Confirm->Approve->Customer Approve->Done->Create Quotation.
- Change Order Appove by Project Manager.
- Allow you to print Change order and Send to your customer for Signature using open chatter.
- Allow you to create quotation directly from Change order.
- Fully integrated with Sales App.
- Change order smart buttons on Project, Job Orders and Customer form.
- For more details about job costing you can see depedant app.
- Watch video in live preview.
