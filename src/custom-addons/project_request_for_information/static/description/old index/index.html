Request for Information

Construction and Contracting Business

This app allow you to manage "Request for Information" for Construction and Contracting Business. Your customer/supplier can submit request online from your website and team can response with answers. Project manager and project team can also manually create request for information from backend in case if they want to ask questions to supplier/customer. This way your project of construction and contracting can get assured information stored well in database.

Main Features

* Your Customer/Supplier can send request from your website and also attach multiple documents while submitting request.
* Generation of unique Request record on submission and record it as Request for Information in backend of Odoo.
* Customer/Supplier can check status of all Request for Information submitted by him/her on My Account page.
* Print PDF - Request for Information
* Project Team / RFI Team can communitcate with Customer/Supplier using Open Chatter and fill Timesheet on Request for Information form in backend.
* RFI Team / Project Manager can close Request for Information.
* Customer/Supplier can give feedback and rating of Request for Information.
* Integrated with Odoo Survey module so for Request for Information - Your team can create survey for that Request for Information.
* Configuration for Request for Information Types.
* Configuration for Request for Information Subjects.
* Configuration for Request for Information Teams.
* Configuration for Request for Information Stages.
* For more details contact us or watch Video.
