Work Package for Construction and Contracting

This app allow you to create and send work package to your customers for your construction and contracting business. For Construction and Contracting please see dependant app to get more details.

Main Features:

  • Work Package creation by adding multiple job order / work order to include in it.
  • Work Package PDF report.
  • Work Package send by email feature.
  • Work Package linked with Project and Job order

Menus Available:

  • Jobs
    • Job Orders
      • Work Packages

Workflow of Work Package

  • Draft ==> Confirmed ==> Sent
