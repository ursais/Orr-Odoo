Timesheets Sheet for Employee

This app allow your employee to fill Timesheet Sheet - Concept is removed from Odoo 11 Standard so this app allow you to have that feature back again. For timesheet from website you can see and buy https://apps.odoo.com/apps/modules/12.0/odoo_timesheet_sheet_web/

Main Features:
  • Allow employee to create and fill Timesheet Sheet.
  • Company settings to set Weekly or Monthly.
  • Workflow of approval of timesheet.
  • Submit by employee and Approved by Manager.
