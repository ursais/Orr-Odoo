Construction and Contracting Issue Tracking Mangement System

This module allow you manage Construction and Contracting Issues. This app is for issue tracking system for Construction and Contracting business. This issue tracking system is for workers who work on site at construction and job contracting and allow them to raise any kind of issues related to jobs they are working on. Your customer can also use this system to raise support tickets.

Main Features

  • Your Customer/Construction Team can send issues request from your website my account / portal page and also attach documents.
  • Generation of unique issue number on submission and record it as issue in backend.
  • Customer/Construction Team can check status of all support issues submitted by him/her on My Account page.
  • Print PDF report of Issue.
  • Issue User / Technician can communitcate using chatter and fill timesheet.
  • Issue Manager can close ticket and send bill to customer if paid support.
  • Customer/Team can give feedback and rating of ticket.
  • Manage your issues tickets using assignment to multiple issue teams.
  • Manage prepaid hours of customer for issues tickets.
  • Create customer invoices from timesheet lines/activities.
  • Add Time In and Time Out on timesheet lines/activities.
  • Added Dynamic Stages for issue management system related to your business and workflow.
  • Added Issue Types.
  • Added Type of Subjects.
