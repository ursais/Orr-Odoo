Mobile Timesheet Odoo

This app build Mobile Timesheet in Odoo which is completly Odoo web based framework. (This is NOT Android and IOS build).

Created Menu:

  • Timesheets/Configuration/Work Types

    • Main Features:

  • Allow your Employee ,Internal Users and Timesheet User to fill Timesheet in Mobile and Tablets.
  • This is completly web based framework in Odoo and fully responsive in all kind of Mobiles and Tablets.
  • Your employee/staff/Internal User/ Timesheet User Employee can fill Timesheet from anywhere like when they are working onsite, From home, From Job Sites, From Customer Location etc etc. And fully integrated with Odoo backend Timesheet.
  • Note: This is NOT Android and IOS/Iphone app.
  • Fully Responsive.
  • Working with Odoo Enterprise as well Odoo Community Editions.
  • Added more fields on Odoo Timesheet in backend like Start Time, End Time, Work type, etc etc..
  • Allow user to make quick search for anytimesheet from Mobile.
  • Allow user to modify and delete existing submitted timesheets.
  • For more details please watch Video.
