Create Progress Billing and Send to Your Customer

- This module Module allow you to create and print progress billing.

Main Features:

  • Allow you to set Total Progress Billing Amount on the Analytic Account / Project.
  • Added Progress Billing Title on Invoice Form and Selection of Project and Auto compuate Progress Billings Amounts.
  • Allow to print progress billing report so you can send to your customer.

Menus Available:

  • Invoicing
    • Configuration
      • Analytic Accounting
        • Analytic Accounts

      • Invoices
